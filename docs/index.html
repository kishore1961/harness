🔬 RAG Benchmark Harness

India Union Budget 2026-27 LLMOps Reference Implementation

A systematic benchmark of RAG configurations answering the question: Which combination of chunking strategy, retrieval method, reranking, and model gives the best quality-to-cost ratio?

Research Question

Across chunking strategies (300/500/800 tokens), retrieval methods (dense / BM25 / hybrid), reranking (on/off), model choice (Llama 3.1 8B / Llama 3.3 70B / Claude Haiku), and prompt strategy (strict / calibrated / citation-forcing) — what configuration maximizes RAG quality at minimal cost, measured by DeepEval and Ragas?

Methodology

Corpus: India Union Budget Speech 2026-27 (63 pages, 62 chunks)
Eval set: 5 curated Q/A pairs with factual ground truth
Embedding model: sentence-transformers/all-MiniLM-L6-v2 (local, free)
Judge LLM: Claude Haiku (DeepEval) + Groq Llama 8B (Ragas)
Metrics: Faithfulness, Answer Relevancy, Contextual Precision, Contextual Recall (DeepEval) + Context Recall, Answer Correctness (Ragas)
